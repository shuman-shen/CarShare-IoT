Setup¶

Master Pi¶

Backend - RESTful API¶

  1. Install virtual environment tool pipenv.

$ pip3 install pipenv

  1. Set up environment variables:

$ export FLASK_APP=master_pi.py
$ export FLASK_ENV=production
$ export My_SQL={your local MySQL password}

  1. Start MySQL deamon process.

$ mysqld

  1. Go to project folder, install required packages, and start backend server.

$ cd {project_root_folder}/pi
$ pipenv install
$ pipenv shell
$ flask run

  1. Access API root path with port 5000.

http://127.0.0.1:5000/

Frontend - React Web App¶

  1. Configure repository to install yarn.

$ curl -sS https://dl.yarnpkg.com/debian/pubkey.gpg | sudo apt-key add -
$ echo "deb https://dl.yarnpkg.com/debian/ stable main" | sudo tee /etc/apt/sources.list.d/yarn.list

  1. Install yarn package manager.

$ sudo apt update && sudo apt install yarn

  1. Go to project folder and install required node packages.

$ cd {project_root_folder}/web
$ yarn install
$ yarn start

  1. Access Master Pi website.

http://127.0.0.1:3000/
